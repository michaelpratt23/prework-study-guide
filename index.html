Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init: Initializes a new Git repository in the current directory. Run this command to start versioning a new project.
  • git clone <repository URL>: Creates a copy of an existing remote repository locally. Use this to download projects from remote repositories like GitHub.
  • git add <file> or git add .: Adds changes in the specified file or all changes in the current directory to the staging area, preparing them to be committed.
  • git commit -m "commit message": Records the changes in the staging area along with a descriptive message about the changes. This creates a new commit in the repository's history.
  • git push <remote> <branch>: Sends local commits to the remote repository, updating the remote branch with your changes. Use this to share your work with others or back it up remotely.
