  • The body element represents the visible content shown to the user.

CSS

  • The style attribute can change the color of a single h1 heading
  • To embed an entire stylesheet directly in the HTML file use a style element in the head element.
  • When using an external style sheet, we link the file using a link element placed in the HTML's head
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init: Initializes a new Git repository in the current directory. Run this command to start versioning a new project.
  • git clone <repository URL>: Creates a copy of an existing remote repository locally. Use this to download projects from remote repositories
  • git add <file> or git add .: Adds changes in the specified file or all changes in the current directory to the staging area
  • git commit -m "commit message": Records the changes in the staging area along with a descriptive message about the changes
  • git push <remote> <branch>: Sends local commits to the remote repository
